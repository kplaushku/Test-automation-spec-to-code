# Contributing to Specto

Thanks for your interest in contributing!

- **Issues and discussions:** open them in this repository's own tracker
  (https://github.com/kplaushku/Test-automation-spec-to-code/issues), not on github/spec-kit.
- **Pull requests:** fork this repository, create a topic branch, and open a PR
  here.
- **Scope:** keep the neutral layers (`specify`, `clarify`, `plan`, `tasks`)
  framework-agnostic. Framework specifics belong only in adapters under
  `presets/test-kit/templates/adapters/<framework>/`.
- **Adding a framework:** add a new adapter directory; do not modify the neutral
  layers or the core orchestration scripts.
