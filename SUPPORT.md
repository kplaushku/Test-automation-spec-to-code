# Support

Need help with Specto?

- Read the docs in [`docs/`](docs/) and the preset guide in
  [`presets/test-kit/README.md`](presets/test-kit/README.md).
- Search existing issues, then open a new one in this repository's tracker
  (https://github.com/kplaushku/Test-automation-spec-to-code/issues).

For questions about the upstream tool itself, see
[github/spec-kit](https://github.com/github/spec-kit). Specto is an
independent fork and is supported separately.
