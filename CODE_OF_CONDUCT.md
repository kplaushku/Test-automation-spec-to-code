# Code of Conduct

Specto follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, assume good intent, and keep discussions constructive.

To report unacceptable behavior, contact the maintainers at
the maintainer @kplaushku via a private GitHub Security Advisory on this repository.

> This project is derived from [github/spec-kit](https://github.com/github/spec-kit)
> but is **not** affiliated with, endorsed by, or sponsored by GitHub, Inc.
> Conduct reports go to the Specto maintainers, not to GitHub.
